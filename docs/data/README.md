# Data Docs

데이터 문서는 DB 스키마, 마이그레이션, 제약 조건, 인덱스를 기록한다.

## 파일 역할

| 파일 | 역할 | 업데이트 기준 |
| --- | --- | --- |
| `schema.md` | 테이블과 컬럼 정의 | 마이그레이션 작성 전후 |
| `constraints-and-indexes.md` | unique, FK, index, check 조건 | 무결성/조회 성능 정책이 바뀔 때 |
| `migration-guidelines.md` | Flyway 마이그레이션 작성 규칙 | 마이그레이션 방식이 바뀔 때 |

## 작성 원칙

- 도메인 문서와 DB 문서는 서로 링크한다.
- DB 제약은 애플리케이션 검증과 별개로 명시한다.
- 마이그레이션 파일명과 문서 변경을 함께 확인한다.
